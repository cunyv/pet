# 🚀 快速入门指南

## 安装应用（5分钟）

### 1. 找到安装包

在 `dist` 目录中有两个版本：

```
dist/
├── Pixel Pet-1.0.0.dmg        # Intel Mac (86MB)
└── Pixel Pet-1.0.0-arm64.dmg  # Apple Silicon Mac (82MB)
```

**如何选择？**
- 2020年后的Mac（M1/M2/M3）→ 选 `arm64` 版本
- 较早的 Intel Mac → 选普通版本

### 2. 安装应用

1. 双击 `.dmg` 文件
2. 将 `Pixel Pet.app` 拖到 `Applications` 文件夹
3. 完成！

### 3. 首次运行

由于应用没有签名，首次运行需要：

1. 打开 `应用程序` 文件夹
2. 找到 `Pixel Pet`
3. **右键点击** -> 选择 `打开`
4. 在弹出的对话框中点击 `打开`

## 🎮 操作说明

### 基础操作

| 操作 | 效果 |
|------|------|
| 🖱️ 按住拖拽 | 移动宠物位置 |
| 👆 单击 | 摸头，宠物开心冒爱心 ❤️ |
| 👆👆 双击 | 喂食，宠物吃东西 🍕 |
| 🖱️ 右键 | 显示操作菜单 |

### 右键菜单功能

- 🍕 **喂食** - 给宠物食物
- 😴 **睡觉** - 让宠物休息
- ⏰ **叫醒** - 叫醒宠物
- 🤚 **摸摸头** - 互动
- ❌ **退出** - 关闭应用

---

## 🎨 宠物状态

宠物有 5 种状态：

1. **待机** 😊 - 左右轻微摇晃
2. **走路** 🚶 - 在桌面随机走动
3. **睡觉** 😴 - 闭眼冒 💤
4. **开心** 😄 - 被摸头后跳跃冒爱心
5. **吃东西** 😋 - 张嘴吃东西

## 🛠️ 从源码运行

### 前置要求

```bash
# 检查 Node.js 是否安装
node --version
npm --version
```

### 安装步骤

```bash
# 进入项目目录
cd pixel-pet

# 安装依赖
npm install

# 启动应用
npm start
```

### 开发模式

```bash
# 带开发者工具启动
npm run dev
```

---

## 📦 重新打包

如果修改了代码，需要重新打包：

```bash
# 生成图标
./scripts/generate-icon.sh

# 打包 Mac 应用
npm run build:mac
```
