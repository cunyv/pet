# 🎮 像素宠物 (Pixel Pet)

一个可爱的像素风格桌面宠物应用，使用 Electron 开发，支持 Mac 系统。

## ✨ 功能特性

- 🎨 **像素风格** - 真正的像素方块小人，复古可爱
- 🖱️ **拖拽移动** - 可以把宠物拖到桌面任意位置
- 🤚 **互动功能** - 点击摸头、喂食、让宠物睡觉
- 🎭 **多种状态** - 待机、走路、睡觉、开心、吃东西
- 💬 **气泡对话** - 宠物会显示可爱的对话气泡
- ✨ **粒子特效** - 摸头时会冒出爱心和星星
- 🍕 **喂食系统** - 双击或右键菜单喂食
- 😴 **自动行为** - 宠物会自己走动、待机、睡觉
- 📱 **系统托盘** - 方便控制和退出应用

## 🚀 快速开始

### 安装应用

1. 下载 `dist` 目录中的 `.dmg` 文件：
   - `Pixel Pet-1.0.0.dmg` - Intel Mac
   - `Pixel Pet-1.0.0-arm64.dmg` - Apple Silicon Mac (M1/M2/M3)

2. 双击 `.dmg` 文件打开

3. 将 `Pixel Pet.app` 拖拽到 `Applications` 文件夹

4. 首次运行：右键点击应用 -> 打开

### 从源码运行

```bash
cd pixel-pet
npm install
npm start
```

## 🎮 使用方法

### 基础操作

| 操作 | 说明 |
|------|------|
| **拖拽** | 按住左键拖拽宠物移动 |
| **单击** | 摸头，宠物会开心冒爱心 |
| **双击** | 喂食，宠物会吃东西 |
| **右键** | 显示菜单选项 |

### 右键菜单

- 🍕 **喂食** - 给宠物食物
- 😴 **睡觉** - 让宠物睡觉
- ⏰ **叫醒** - 叫醒睡觉的宠物
- 🤚 **摸摸头** - 互动
- ❌ **退出** - 关闭应用

### 系统托盘

点击菜单栏中的宠物图标可以：
- 显示/隐藏宠物
- 喂食、睡觉、叫醒
- 退出应用

## 🎨 宠物状态

| 状态 | 图标 | 说明 |
|------|------|------|
| 待机 | 😊 | 左右轻微摇晃 |
| 走路 | 🚶 | 在桌面随机走动 |
| 睡觉 | 😴 | 闭眼，头上冒 💤 |
| 开心 | 😄 | 被摸头后跳跃冒爱心 |
| 吃东西 | 😋 | 张嘴吃东西动画 |

## 🛠️ 自定义

### 修改宠物外观

编辑 `src/styles/main.css` 文件可以修改：
- 像素颜色（`.px.hair`, `.px.skin`, `.px.cloth` 等）
- 动画效果
- 窗口大小

### 添加新动画

在 `src/scripts/animation.js` 中可以添加新的粒子效果。

### 修改行为逻辑

在 `src/scripts/pet.js` 中可以修改：
- 自动行为逻辑
- 状态转换规则
- 属性变化速度

## 📦 打包构建

### 生成应用图标

```bash
./scripts/generate-icon.sh
```

### 打包 Mac 应用

```bash
# 打包所有架构
npm run build:mac

# 仅打包 Intel Mac
npm run build:mac:x64

# 仅打包 Apple Silicon Mac
npm run build:mac:arm64
```

打包完成后，`.dmg` 文件会生成在 `dist` 目录中。

## 📁 项目结构

```
pixel-pet/
├── main.js              # Electron 主进程
├── preload.js           # IPC 通信
├── src/
│   ├── index.html       # 主页面
│   ├── styles/
│   │   └── main.css     # 像素风格样式
│   └── scripts/
│       ├── pet.js       # 宠物核心逻辑
│       ├── animation.js # 动画控制器
│       └── interaction.js # 交互管理
├── resources/
│   └── icon.icns        # 应用图标
└── dist/                # 打包输出
```

## 🐛 常见问题

### Q: 应用无法启动？

A: 
1. 右键点击应用 -> 打开
2. 系统偏好设置 -> 安全性与隐私 -> 允许打开

### Q: 宠物不显示？

A: 尝试重启应用，检查是否有其他窗口遮挡。

### Q: 如何让宠物开机自启？

A: 系统偏好设置 -> 用户与群组 -> 登录项 -> 添加 Pixel Pet
