# 📋 更新日志

## v1.2.0 (2026-06-24)

### ✨ 新功能
- 增加鼠标穿透模式开关，可从右键菜单开启、从托盘菜单切换
- 右键菜单新增隐藏宠物入口
- 右键菜单新增透明度、大小、窗口置顶、自动隐藏和重置位置设置
- 增加更自然的宠物状态机，支持无聊、困、饿、开心、委屈和跟随鼠标状态
- 宠物会根据心情、精力、饥饿度和无聊度自动切换状态
- 无聊或被鼠标吸引时，宠物会尝试跨桌面跟随鼠标

### 🎨 样式优化
- 优化右键菜单布局，增加标题、分组、图标列和危险操作高亮
- 穿透模式菜单项会显示当前开启/关闭状态
- 右键菜单内容过高时支持滚动，避免设置项被截断
- 为无聊、困、饿、委屈和跟随状态增加专属动画表现

## v1.1.1 (2026-06-24)

### 🐛 修复
- 优化宠物鼠标碰撞边界，只让可见像素、气泡、动画和右键菜单响应鼠标
- 修复透明窗口空白区域挡住桌面和其他应用操作的问题
- 拖拽结束后自动恢复透明区域鼠标穿透，减少误拦截

## 版本号说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
